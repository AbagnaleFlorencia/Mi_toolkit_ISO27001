# ISO/IEC 27001 & ISO 31000 SGSI Toolkit

> Toolkit desarrollado para apoyar la implementación y gestión de un Sistema de Gestión de Seguridad de la Información (SGSI),
> integrando los requisitos de **ISO/IEC 27001:2022** con la metodología de gestión de riesgos de **ISO 31000**.

---

## Objetivo

Este proyecto nace con el propósito de centralizar en una única herramienta las principales actividades que se realizan durante la implementación de un SGSI.

En lugar de utilizar múltiples documentos independientes, el toolkit organiza de forma estructurada el ciclo completo de gestión:

- Contexto organizacional
- Política de Seguridad
- Inventario de Activos
- Identificación de Riesgos
- Evaluación de Riesgos
- Plan de Tratamiento de Riesgos (RTP)
- Statement of Applicability (SoA)
- Roadmap de implementación
- Dashboard Ejecutivo

La herramienta está pensada como material de apoyo para proyectos académicos, laboratorios de práctica y simulaciones de implementación de ISO/IEC 27001.

---

# Funcionalidades

## Contexto Organizacional

Registro de:

- Partes interesadas
- Necesidades y expectativas
- Alcance del SGSI
- Factores internos y externos

---

## Política de Seguridad

Plantilla para documentar la política de seguridad de la organización.

---

## Inventario de Activos

Permite registrar activos de información incluyendo:

- Propietario
- Clasificación
- Confidencialidad
- Integridad
- Disponibilidad
- Criticidad
- Ubicación
- Estado

---

## Identificación de Riesgos

Registro estructurado de:

- Activo
- Amenaza
- Vulnerabilidad
- Riesgo identificado
- Consecuencia

---

## Evaluación de Riesgos

Matriz para evaluar:

- Probabilidad
- Impacto
- Nivel de riesgo
- Riesgo inherente
- Riesgo residual

basada en la metodología propuesta por ISO 31000.

---

## Plan de Tratamiento (Risk Treatment Plan)

Permite documentar:

- Riesgo tratado
- Control seleccionado
- Responsable
- Fecha objetivo
- Estado
- Evidencias

---

## Statement of Applicability (SoA)

Plantilla para documentar la aplicabilidad de los controles de **ISO/IEC 27001:2022 Anexo A**, indicando:

- Control
- Aplicabilidad
- Justificación
- Estado
- Evidencia

---

## Roadmap de Implementación

Cronograma para planificar la implementación del SGSI.

Incluye:

- Fases
- Prioridades
- Seguimiento
- Estado

---

## Dashboard Ejecutivo

Dashboard orientado a la dirección con indicadores de alto nivel para realizar seguimiento del avance del SGSI.

Ejemplos de indicadores:

- Riesgos identificados
- Riesgos críticos
- Activos inventariados
- Controles implementados
- Estado del proyecto

# Relación con ISO

Este toolkit fue desarrollado tomando como referencia:

- ISO/IEC 27001:2022
- ISO 31000:2018

No pretende reemplazar la documentación oficial de estas normas ni constituye una metodología certificada de implementación.
